# Cloud-Based Firewall for Website Protection

## Project Overview
This project demonstrates the design and configuration of a cloud-based firewall using Cloudflare to protect a live website. It covers firewall rules, WAF (Web Application Firewall), attack monitoring, and comprehensive documentation of cloud-based web protection.
